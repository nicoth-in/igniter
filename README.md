# Igniter 0.3.0-dev
Ignite your pages! Igniter is a *frontend* JS engine with zero-cost abstraction.
It helps you to build and manage DOM in an object-oriented programming style.

## Dev
Igniter v0.3.x is built on top of the DOM.
